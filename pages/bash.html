Bash Cheat Sheet

Navigation

pwd Print working directory
cd <dir> Change directory
cd .. Move up one directory
cd ~ Go to home directory
cd - Go to previous directory
pushd <dir> Save current dir and move to another
popd Return to saved directory

File Operations

ls -lah List files with details, hidden, human-readable
touch <file> Create empty file or update timestamp
cp <src> <dest> Copy file
cp -r <src> <dest> Copy directory recursively
mv <src> <dest> Move or rename file
rm <file> Remove file
rm -rf <dir> Remove directory recursively destructive
mkdir -p <path/to/dir> Create directory with parent directories
rmdir <dir> Remove empty directory

File Viewing

cat <file> Display entire file
less <file> View file with scrolling better than more
head -n 10 <file> Show first 10 lines
tail -n 10 <file> Show last 10 lines
tail -f <file> Follow file updates in real time for logs
wc -l <file> Count lines in file

Search & Find

grep "pattern" <file> Search for pattern in file
grep -r "pattern" <dir> Search recursively in directory
grep -i "pattern" <file> Case-insensitive search
grep -v "pattern" <file> Invert match — show non-matching lines
find <dir> -name "*.txt" Find files by name pattern
find <dir> -type f -mtime -7 Find files modified in last 7 days
locate <filename> Fast search using database
which <command> Show path of executable

Permissions

chmod 755 <file> Set rwxr-xr-x permissions
chmod +x <file> Make file executable
chmod -R 644 <dir> Set permissions recursively
chown <user>:<group> <file> Change owner and group
chown -R <user> <dir> Change owner recursively
sudo <command> Run command as root
sudo -i Open root shell

Processes

ps aux List all running processes
ps aux | grep <name> Find specific process
top Real-time process viewer
htop Better interactive process viewer
kill <pid> Terminate process by ID
kill -9 <pid> Force kill process destructive
killall <name> Kill all processes by name
pkill <pattern> Kill processes matching pattern
<cmd> & Run command in background
jobs List background jobs
fg %1 Bring job 1 to foreground

Pipes & Redirection

<cmd1> | <cmd2> Pipe output to another command
<cmd> > <file> Redirect output to file (overwrite)
<cmd> >> <file> Append output to file
<cmd> 2> <file> Redirect stderr to file
<cmd> &> <file> Redirect both stdout and stderr
<cmd> < <file> Use file as input
<cmd> | tee <file> Write to file and stdout simultaneously

Text Processing

sort <file> Sort lines alphabetically
sort -n <file> Sort numerically
uniq <file> Remove duplicate adjacent lines
sort <file> | uniq Remove all duplicates
cut -d ',' -f 1,3 <file> Extract columns 1 and 3 from CSV
awk '{print $1}' <file> Print first field of each line
sed 's/old/new/g' <file> Replace all occurrences of old with new
tr 'a-z' 'A-Z' < <file> Convert lowercase to uppercase

Archiving

tar -czf <archive.tar.gz> <dir> Create compressed tar archive
tar -xzf <archive.tar.gz> Extract compressed tar archive
tar -tzf <archive.tar.gz> List contents without extracting
zip -r <archive.zip> <dir> Create zip archive
unzip <archive.zip> Extract zip archive
unzip -l <archive.zip> List zip contents

System Info

uname -a Show system information
hostname Display system hostname
uptime Show how long system has been running
df -h Show disk usage, human-readable
du -sh <dir> Show directory size
free -h Show memory usage
lsblk List block devices
lscpu Display CPU info

Network

ping <host> Test connectivity to host
curl <url> Fetch URL content
curl -O <url> Download file from URL
wget <url> Download file
ip addr Show network interfaces
netstat -tuln Show listening ports
ss -tuln Modern alternative to netstat faster
scp <file> <user>@<host>:<path> Secure copy to remote host

Variables & Environment

VAR="value" Set variable
echo $VAR Print variable value
export VAR="value" Set environment variable
env Show all environment variables
printenv VAR Print specific environment variable
unset VAR Remove variable
source <file> Execute file in current shell

Aliases & History

alias ll='ls -lah' Create command alias
alias List all aliases
unalias ll Remove alias
history Show command history
!n Execute command number n from history
!! Execute last command
sudo !! Re-run last command as root
Ctrl+R Reverse search command history

Scripting Basics

#!/bin/bash Shebang — first line of script
if [[ $VAR == "value" ]]; then ... fi Conditional statement
for i in {1..10}; do echo $i; done For loop
while [[ $VAR -lt 10 ]]; do ... done While loop
function name() { ... } Define function
$1, $2, $@ Positional parameters — script arguments
$? Exit status of last command
exit 0 Exit script with status code
